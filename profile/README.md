## 🌍 What We're Building

> Just-Code-Builder is the engineering organisation behind **K-Lab** — a portfolio of 7 interconnected apps providing the digital rails for Africa's next billion users.
>
> Every product shares a single backend identity layer (**Kernal**), a single wallet system, KYC, and push notification infrastructure. One account. Every app.

## 🏗️ Product Portfolio


<table>
<tr>
<td width="" valign="top">

### 🔵 Klix — Web3 Social Super-App
**X:** [@Klix Social](https://x.com/ZyndrHQ)

Decentralised social layer for Africa. Wave algorithm, creator economy, live commerce integration, Web3 token model. The social discovery layer that connects all K-Family apps.

**Stack highlights:** React Native · Node.js · Rust · Kafka · DeFi

---

### 🟠 Krat — Pan-African Marketplace
**X:** [@AddKrat](https://x.com/AddKrat)

Super-marketplace combining second-hand & new goods, food ordering, groceries, live selling, auctions, group buying, and a GPS-dispatched rider network. Africa's answer to Amazon + Shopify + Glovo.

**Stack highlights:** React Native · Node.js · TypeScript · Google Maps API · Paystack

---

### 🟡 CloutIx — Engagement Exchange
**X:** [@CloutIx](https://x.com/CloutIx) &nbsp;🚀 **Live**

Manual social media task marketplace — buyers get real engagement, earners get paid. Five-level referral structure, influencer tiers, premium membership, multi-account declaration system.

**Stack highlights:** React Native CLI · Node.js · PostgreSQL · Kernal Auth

</td>
<td width="50%" valign="top">

### 🟢 Kribs — Housing Rental Platform
**X:** [@UseKrib](https://x.com/UseKrib)

Trust-first GPS-verified housing rental. NIN/BVN verified landlords, full rent held in Paystack DVA escrow until move-in confirmed. Inspection after payment. Refund if tenant declines.

**Stack highlights:** React Native · Node.js · Dojah KYC · Paystack DVA · Google Maps

---

### 🟣 Kredio — Crypto Payment Gateway
**X:** [@GetKredio](https://x.com/GetKredio)

Crypto payment gateway targeting merchants across Africa, SE Asia, Latin America, and the Middle East. Pool wallet, HD-derived payment addresses, DeFi trading layer (NexPump). Public global API.

**Stack highlights:** Java Spring Boot · Rust · Node.js · HAProxy · HD Wallets

---

### 🔴 KITE — DePIN VPN
**Status:** 📐 Planned

Decentralised VPN + bandwidth marketplace. Node sellers earn by sharing bandwidth; consumers buy secure, private internet access. Tiered: Free · Premium Shared · Premium Dedicated · Residential IP.

**Stack highlights:** Rust · Go · DePIN Protocol · Node.js

## 🏛️ Architecture Principles

```
┌─────────────────────────────────────────────────────────┐
│                    K-LAB ECOSYSTEM                   │
├─────────────────────────────────────────────────────────┤
│   Klix    │   Krat    │  Kribs  │ Kredio │ KITE │CloutIx│
├─────────────────────────────────────────────────────────┤
│                 KERNAL CORE API                         │
│        Auth · Wallet · KYC · Push · Identity            │
├──────────────┬──────────────┬───────────────────────────┤
│   Neon DB    │  Upstash     │   Apache Kafka            │
│ (PostgreSQL) │  (Redis)     │   (Event Streaming)       │
├──────────────┴──────────────┴───────────────────────────┤
│  Rust (Perf) │  Go (Gateway) │  Python FastAPI (AI/ML)  │
└─────────────────────────────────────────────────────────┘
```

- **One account** across all 7 apps (phone number = identity)
- **One wallet** — main balance in Kernal; app earnings withdraw via `/wallet/credit`
- **Event-driven** — Kafka streams tie apps together in real time
- **Performance-critical paths** in Rust; concurrent gateway tasks in Go
- **AI layer** — Python microservices for fraud detection, recommendations, matching
